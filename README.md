# Chess Engine

## 소개
텍스트 기반 체스 게임 시스템

## 기능
- 체스 기물 이동

## 구동
python main.py

## 기술
python

## 구조
main.py   : 게임 실행
board.py  : 체스 보드 관리
pieces.py : 기물 이동 규칙

## 개발 목적
체스 기반 이동 시스템을 연습하기 위한 개인 프로젝트입니다.
기본 체스 기능 완성후 이 코드를 베이스로 다양한 창작 게임 규칙을 실험하는 것을 목표로 합니다.

## 로드맵

- 체크 / 체크메이트 판정
- 게임 종료 조건
- AI 상대
